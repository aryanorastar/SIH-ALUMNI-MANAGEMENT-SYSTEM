# SIH Alumni Management System

A comprehensive web-based platform designed to facilitate alumni engagement, networking, and community building. This system provides a unified space for alumni to reconnect, share opportunities, and stay informed about institutional updates.

## 🎯 Project Overview

The SIH Alumni Management System is a feature-rich application developed to enhance alumni engagement and networking. It serves as a central hub where alumni can manage their profiles, connect with peers, and participate in various community activities.

## ✨ Key Features

### 📱 **User Management**
- Secure login and user authentication
- Customizable user profiles with career information
- Alumni directory with advanced search capabilities
- Profile visibility and privacy controls

### 🤝 **Networking & Community**
- Alumni profiles browsing and discovery
- Campus feed for community interactions
- Messaging system for direct alumni-to-alumni communication
- Networking opportunities and connections

### 📢 **Communication Hub**
- Announcements section for institutional updates
- Newsletter management and distribution
- Event announcements and details
- Job board for career opportunities
- Direct messaging between alumni members

### 📅 **Events & Activities**
- Event management and registration
- Campus feed for sharing updates and memories
- Memories section for alumni nostalgia and engagement
- Alumni gatherings and reunion coordination

### 💼 **Career Development**
- Job board featuring alumni-posted opportunities
- Career networking features
- Professional opportunity sharing

### 📰 **Information & Updates**
- Newsletters for regular communication
- Announcements about institutional news
- About section with platform information
- Contact information and support

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or server setup required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aryanorastar/SIH-ALUMNI-MANAGEMENT-SYSTEM.git
cd SIH-ALUMNI-MANAGEMENT-SYSTEM
```

2. Open the application:
   - Simply open `index.html` in your web browser
   - Or deploy the files to a web server

### Basic Usage

1. **First Visit**: Start at `index.html` to access the homepage
2. **Login**: Navigate to `login.html` to create an account or sign in
3. **Profile Setup**: Complete your profile on `user-profile.html`
4. **Explore**: Browse alumni profiles, events, and opportunities
5. **Engage**: Post to campus feed, message alumni, and participate in community
